# Progetto Unipegaso traccia 15
Simulazione comparativa tra una azienda di agricoltura e allevamento sostenibile e una intensiva. 

## STRUTTURA
```
├── main.py        # Avvia la simulazione
├── funzioni.py    # Funzioni di calcolo e output
├── config.py      # Variabili e parametri globali delle aziende
├── requirements.txt # Necessari per il codice
└── README.md
```

## COME ESEGUIRE
```bash
pip install requirements.txt
python main.py
```

## OUTPUT
* tabella di comparazione produzione costi e tempistiche delle due aziende
* grafici a barre per un confronto visuale di questi tre output
* report_azienda.pdf viene salvato nella cartella del progetto. file pdf con la tabella di comparazione
